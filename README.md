# 🧾 LINE Bot บันทึกค่าขนส่ง (โรงเย็บผ้าม่าน)

ถ่ายรูปบิลขนส่งส่งใน LINE → บอทอ่านข้อมูลให้ → ยืนยัน/แก้ไข → บันทึกลง Google Sheet → สรุปค่าขนส่ง **แยกตามสาขา** เพื่อเบิกเงินแต่ละสาขาได้ทันที

**เทคโนโลยี (ฟรีทั้งหมด ไม่ต้องผูกบัตร):** Node.js · LINE Messaging API · OCR.space (อ่านตัวอักษรไทย) · Groq Llama (จัดข้อมูล) · Google Sheets · Render

---

## ข้อมูลที่เก็บ (คอลัมน์ในชีท)

วันที่ · ร้านสาขา · เจ้าของงาน · ชื่องาน · บริษัทขนส่ง · เลขที่บิล · รายการสินค้า · จำนวน(ชิ้น) · ค่าขนส่ง(บาท) · วิธีเก็บเงิน
(+ เวลาบันทึก, รหัสบิล อัตโนมัติ)

บอทจะอ่านจากบิลให้ก่อน ช่องที่อ่านไม่เจอ (สาขา/เจ้าของงาน/ชื่องาน) จะเว้นว่างให้พิมพ์เติม — ทุกช่องแก้ได้

---

## เมนู 3 ปุ่ม (Rich Menu) — ขึ้นเองอัตโนมัติหลัง deploy

1. **📊 รวมยอด** → เลือกช่วงเวลา (เดือนนี้/เดือนที่แล้ว/7 วัน/ระบุเอง) → เลือก รวมทั้งหมด / แยกตามสาขา / แยกตามขนส่ง
2. **🗂️ เปิดชีท** → เปิด Google Sheet (ดาต้าเบสทั้งหมด)
3. **📸 ถ่ายรูปบิล** → เปิดกล้องถ่ายบิลส่งเข้ามาได้เลย

> ถ้าแก้ไขเมนูภายหลัง (เปลี่ยนรูป/ปุ่ม) ให้ตั้ง env `REBUILD_RICHMENU=1` แล้ว deploy ใหม่ 1 ครั้ง จากนั้นเอาออก

---

## วิธีใช้งานในแชท

| พิมพ์/ทำ | ผลลัพธ์ |
|---|---|
| ส่งรูปบิล | บอทอ่านให้ตรวจ พร้อมปุ่มยืนยัน/แก้ไข |
| `รวมยอด` | เริ่มสรุป (เลือกช่วง → กลุ่ม) |
| `สรุปเดือนนี้ แยกสาขา` | สรุปทางลัด (พิมพ์ครั้งเดียว) |
| `สรุป 1/6/2026 ถึง 15/6/2026 แยกขนส่ง` | สรุปช่วงวันที่ แยกบริษัทขนส่ง |
| `เปิดชีท` | ลิงก์เปิด Google Sheet |
| `บิลล่าสุด` | ดู 5 บิลหลังสุด |
| `เมนู` | วิธีใช้ |

**แก้ข้อมูลบิล** (ตอนตรวจ พิมพ์ได้หลายอันคั่นด้วยขึ้นบรรทัด/คอมม่า):
```
สาขา=ยะลา
เจ้าของงาน=ซิลมี
ชื่องาน=ผ้าม่านห้องนอน
ค่าขนส่ง=150
ขนส่ง=ไชยรักษ์เอ็กซ์เพรส
จำนวน=1
วิธีเก็บเงิน=ปลายทาง
```

## ติดตั้ง (ครั้งแรก)

ดูขั้นตอนเดิมที่เคยทำ: สมัคร LINE Messaging API, ขอคีย์ OCR.space + Groq, สร้าง Google Sheet + Service Account (แชร์ชีตให้อีเมล service account เป็น Editor), แล้ว deploy ขึ้น Render

**Environment Variables ที่ต้องตั้งใน Render**

| Key | ค่า |
|---|---|
| `LINE_CHANNEL_ACCESS_TOKEN` | จาก LINE |
| `LINE_CHANNEL_SECRET` | จาก LINE |
| `OCR_SPACE_API_KEY` | จาก ocr.space |
| `OCR_SPACE_LANGUAGE` | `auto` |
| `OCR_SPACE_ENGINE` | `2` |
| `GROQ_API_KEY` | จาก console.groq.com |
| `GROQ_MODEL` | `llama-3.3-70b-versatile` |
| `GOOGLE_SHEET_ID` | ID ของชีท (ใช้ทำลิงก์ปุ่มเปิดชีทด้วย) |
| `GOOGLE_SHEET_NAME` | `Bills` |
| `GOOGLE_SERVICE_ACCOUNT_JSON` | เนื้อหาไฟล์ JSON ทั้งก้อน |
| `CLOUDINARY_CLOUD_NAME` | (ไม่บังคับ) เก็บรูปบิล — จาก cloudinary.com |
| `CLOUDINARY_API_KEY` | (ไม่บังคับ) |
| `CLOUDINARY_API_SECRET` | (ไม่บังคับ) |
| `LIFF_ID` | จากการสร้าง LIFF app |
| `TZ` | `Asia/Bangkok` |
| `REBUILD_RICHMENU` | (ไม่บังคับ) ใส่ `1` เมื่ออยากสร้างเมนูใหม่ |

## หมายเหตุ

- Render free tier จะ "หลับ" เมื่อไม่มีคนใช้ บิลใบแรกหลังหลับอาจช้า ~30 วิ
- บิลที่ยังไม่ยืนยันเก็บชั่วคราวในหน่วยความจำ ถ้าเซิร์ฟเวอร์รีสตาร์ทต้องส่งรูปใหม่ (ที่บันทึกแล้วอยู่ในชีตปลอดภัย)
- ถ้าอ่านไทยเพี้ยน กดแก้ไขในแชทได้เสมอ
